Service Provider Examples

This folder contains examples of how to implement a Service Provider to communicate between TIM and a target resource.  The Connector API is used for implementing communication protocols with endpoints.  There are examples for both outgoing and incoming communication.  

Outgoing communication:
  • File Service Provider
  • JNDI (LDAP) Service Provider
  • Relational Database Service Provider
Incoming notification
  • Event notification
There is also an example of communicating with TIM via the Directory Service Markup Language version 2 (DSMLv2) Service Provider using a Java Naming and Directory Interface (JNDI) gateway, which is described in the document Directory Service Markup Language Version 2 JNDI Example.
